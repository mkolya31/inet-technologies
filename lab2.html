Лабораторная работа №2

Разработка IT проекта в команде
К содержимому


Цель командного проекта - создание единого Android приложения для МГТУ Станкин с расписанием занятий, модульным журналом, актуальными новостями и информацией об университете
  • В ходе выполнения лабораторной работы были решены следующие задачи:
  • 1. Разработана концепция Android приложения StankinINFO
  • 2. Создан репозиторий приложения
  • 3. Разработана архитектура приложения
  • 4. Определены и интегрированы основные используемые библиотеки
  • 5. Совместно с Алексеем Балаганским производилась разработка основных функций приложения
Описание Android приложения

Приложение было разработано в среде разработки Android Studio на языке программирования Kotlin. В приложении используется современная архитектура MVVM. Для создания приложения также применялись библиотеки Android Jetpack, RxJava2, Retrofit, Dagger, Glide, Tink.

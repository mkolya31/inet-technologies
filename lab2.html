Лабораторная работа №2

Разработка IT проекта в команде
К содержимому


Цель командного проекта - создание единого Android приложения для МГТУ Станкин с расписанием занятий, модульным журналом, актуальными новостями и информацией об университете. Ответственность в проекте - создание презентационного веб-сайта приложения, создание презентации.
  • В ходе выполнения лабораторной работы были решены следующие задачи:
  • 1. Разработана концепция презентационного веб-сайта StankinINFO
  • 2. Создан репозиторий веб-сайта
  • 3. Разработана архитектура и дизайн веб-сайта
  • 4. Определены и интегрированы основные используемые библиотеки
  • 5. Подготовлен графический материал веб-сайта
  • 6. Разработан и развёрнут адаптивный веб-сайт приложения StankinINFO
Описание Android приложения

Приложение было сконфигурировано при помощи утилиты npm react app и разрабатывалось с использованием JavaScript фреймворка React. В ходе разработки была использована графическая библиотека Ant Design. В целях увеличения аудитории веб-сайт был полностью адаптирован для использования на устройствах с маленькой шириной экрана (вплоть до 320px) и переведён на английский язык.
